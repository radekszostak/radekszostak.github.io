Calibration using PyTorch

About

This is an unconventional application of PyTorch for calibration of multiple overlapping aerial thermal images using gradient descent. In this task gradient descent optimization is performed twice: for image position alignment and correction of temperature measurements.

Position alignment

Drone GPS and IMU logging allows for georeferencing of aerial photo. However, due to GPS and IMU uncertainties georeferencing is not precise, which results in misalignments in photos mosaics.

Image mosaics can be aligned in post processing. Each pair of overlapping photos is aligned (stithed) using SIFT algorithm provided by OpenCV library. After acquisition of relative positions between each pair of photos gradient descent is used to obtain positions of all photos simultainously which is consensus of all aforementioned pair alignments.

During gradient descent optimization parameters of images transformation matrices are tuned. Loss function minimizes differences between transformation matrices obtained in pair alignment with transformation matrices tuned during optimization.

Temperature correction

Temperatures measurements made using thermal cameras are prone to biases related to cameras internal component temperature changes. This results in notable differences in temperature measurements of the same object visible on two overlapping thermal images. Gradient descent optimization was used to minimize the difference in temeprature measurements between overlapping images for all images at once. Tunable parameter was bias assigned to each photo and loss function was constructed to minimize temperature difference between overlapping parts of images.
